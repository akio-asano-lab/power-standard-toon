# 使い方

## LLMにtoon形式データを認識させる

以下のプロンプトを入力してLLMにtoon形式データを理解させる。

```prompt
あなたはtoon形式データを理解していますか？
理解していないのであればtoon形式データが何かを調査して理解してください。
```

## 電力本位制を理解させる

ElectricPowerStandard_Concept_v1_0.toonとElectricPowerStandard_W_total_v1_0.toonの内容をLLMに入力して電力本位制を理解させる。

## 電力本位制の観点で調査させる

プロンプト例1：

```prompt
日本の30年国債利回りが2025年11月21日に3.41%に到達したことについて、
電力本位制の観点で評価して。
```

プロンプト例2：

```prompt
過去のオイルショック事件について電力本位制の観点で評価して。
```
